- Terminal: [foot](https://codeberg.org/dnkl/foot/)

# Keymap intuition
- `hjkl` for direct navigation, `tab`/`S-tab` for next/prev navigation
- `super` for inter-window, `ctrl` for inner-window, `shift` 
- `f`(find) for sneaking (vim/vimium)
